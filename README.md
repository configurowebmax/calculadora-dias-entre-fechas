# Calculadora de Días Entre Fechas Gratis

Aplicación web gratuita para **calcular la diferencia entre dos fechas** y sumar/restar días.

## ✨ Características

- Diferencia en **días, semanas, meses y años**.
- **Sumar o restar días** a una fecha.
- Formato de fecha largo en español.

## 🚀 Uso

1. Selecciona fecha inicial y final.
2. Pulsa **Calcular diferencia**.
3. O usa la herramienta para sumar/restar días.
